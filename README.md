# CityFlow Flutter

CityFlow is a Flutter mobile app for navigating and discovering services in Redemption City, Ogun State, Nigeria.

The scaffold follows `CODEX.md`:

- Feature folders live in `lib/features/`.
- Shared models live in `lib/shared/models/`.
- Feature-specific models stay inside their feature folder.
- Static seed data lives in `assets/data/`.
- Routes are named in `lib/core/router.dart`.
